# Pikmin Postcards

Pikmin Postcards is a cute community website where players can explore, share, and manage postcard locations on a live map.

Users can browse postcards publicly, view details, vote, and report wrong locations. Signed-in users can upload postcards, use AI location detection, edit their own content, and manage their personal dashboard.
